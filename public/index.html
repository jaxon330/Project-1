Learn to play Simon game

Simon game is a game of lights and sounds in which player must repeat random sequences of lights by pressing the colored pads in correct order

Press any key to start the game. Simon will give the first signal. Repeat the signal by clicking the same color lens. Simon will dublicate color as you advance to next level. Repeat the colors. Continue playing as long as you can repeat each sequence of signals correctly.

If you fail to repeat a sequence exactly, Simon responds with a 'BAZZ' sound. This means you've lost. Game Over!
